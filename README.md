# Quiz Application (Spring Boot)

This is a full-stack Quiz Application built using Spring Boot.

## Features
- Dynamic questions from database
- Timer-based quiz
- Score calculation
- Leaderboard system
- Attractive UI

## Technologies Used
- Java
- - Spring Boot
- Thymeleaf
- H2 Database
- HTML, CSS, JS

## How to Run
1. Clone the repo
2. Open in IDE
3. Run QuizappApplication.java
4. Open http://localhost:8082/quiz
